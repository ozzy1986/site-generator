Генератор киберспортивного сайта

Собирает статический сайт на основе данных PandaScore и обновляет страницы за вчера, сегодня и завтра.

Открыть публичный сайт

Что делает сборка

  • Обновляет данные матчей из PandaScore.
  • Создаёт главную страницу и разделы за три дня.
  • Пересобирает готовый статический сайт для Apache.

Что создаётся

  • Главная страница с обзором по дням.
  • Страницы `вчера`, `сегодня`, `завтра`.
  • SEO-мета, Open Graph и JSON-LD.

Источник данных

Данные матчей загружаются с PandaScore API.
